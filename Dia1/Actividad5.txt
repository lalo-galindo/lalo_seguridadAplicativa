¿Qué es una red?
Un conjunto de dispositivos interconectados

Modelo TCP/IP
Un modelo que ayuda a mostrar las etapas en las que se hace una conexión TCP

Ip
Dirección lógica que diferencia dispositivos en la red

Protocolo TCP
Protocolo de conexión entre dispositivos conocido por comenzar con un proceso de tres pasos conocido como 3 way handshake en el que se establece una sesión antes de mandar la información

HTTP
Hyper text transfer protocol envía texto entre dispositivos, es el que se usaba en las páginas de internet, ahora se usa HTTPS que es su sucesor pero a diferencia de HTTP se envía la información cifrada

HTTP 80
HTTPS 443

HTTPS está cifrado, la S significa Secure

Modelo Cliente- servidor
Modelo en el que un dispositivo (servidor) envía información a los clientes, que la solicitan por medio de requests

Petición
Una solicitud de información que hace un cliente a un servidor, está compuesta de headers e información que indican el tipo de respuesta que se espera

Response
La información que envía el servidor como respuesta a una request, esta tiene diferentes códigos que indican si la request fue exitosa, si no lo fue, entre otros posibles resultados
